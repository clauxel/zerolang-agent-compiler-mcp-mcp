# Architecture Notes

This page explains ZeroLang Agent Compiler MCP from a public developer and evaluator perspective. It does not document private production code.

## MiroFish-Style Documentation Pattern

This repository mirrors the public-facing shape of https://github.com/clauxel/MiroFish: README first, guide pages second, references and safety boundaries close by, and no private runtime material.

## Layers

1. Public explanation layer: homepage, guides, pricing, checkout, and support entry points.
2. Remote MCP layer: endpoint, server card, tool list, and bearer-token access.
3. Evidence layer: reports, receipts, verdicts, or exports that make the workflow auditable.
4. Commercial layer: tracked product links, pricing, checkout, and support.
