# Security

ZeroLang Agent Compiler MCP is designed as a hosted remote MCP with paid access.

## Boundaries

- Bearer token authentication is required for production calls.
- The public documentation repository does not include production secrets or private source code.
- Directory submissions should reference this repository, the product website, and the public server card.
- Do not paste API keys into GitHub issues or public directory forms.

## Reporting

Use the issue template in this repository for public documentation problems. For account, billing, or token issues, use the support contact shown on the product website.
